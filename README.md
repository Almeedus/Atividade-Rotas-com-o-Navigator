# Atividade Rotas com o Navigator
Resposta da Atividade de Rotas com Navigator da disciplina de Desenvolvimento Mobile. A estrutura do arquivo está separada em partes por um comentário informando a qual tela aquele código pertence.

Obs.: Devido a defasagem de hardware do desenvolvedor, foi utilizado o site Dartpad para criar os arquivos e lógicas. Para salvar no github foi copiado e colado o que foi desenvolvido num arquivo com extensão ".dart", e por isso a estrutura de pastas está num formato diferente do que se encontra num projeto de flutter iniciado na máquina.
